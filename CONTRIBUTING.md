# Contributing

Thanks for your interest in contributing to StarryCrypt.

## Getting started

1. Fork the repository
2. Create a feature branch
3. Install dependencies with `npm install`
4. Make focused changes
5. Run:
   - `npm run test`
   - `npm run build`
   - `npm run lint` (note existing baseline lint issues may be unrelated)
6. Open a pull request with a clear summary

## Contribution guidelines

- Keep changes minimal and scoped
- Preserve security properties of cryptographic flows
- Avoid introducing unsafe rendering or script execution paths
- Add or update documentation when behavior changes
