# Code of Conduct

## Our pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive
- Assume good intent
- Focus on technical issues, not personal attacks
- Accept feedback gracefully

## Unacceptable behavior

- Harassment, trolling, or discriminatory language
- Personal attacks or intimidation
- Publishing private information without consent

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove or edit contributions or comments that violate it.
